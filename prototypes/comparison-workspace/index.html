Review note — the printable document. This is the canonical print surface. Printing is the browser's own Save-as-PDF (needs no contract, matching the built CS page); a platform-generated file export stays gated as ESC-CS-EXPORT (the Excel button is a disabled stub). Pages are composed app-side with a real "Page N of M" — no browser page-guessing. The reference reads "Draft" only — no CS-#### series exists yet (ESC-CS-REF). Signatures are wet-ink placeholders — no digital-approval capture (ESC-CS-DOCKIND). Letterhead is a plain placeholder (ESC-CS-LETTERHEAD). Buyer evaluation text carries a † "buyer-authored" mark; the platform authors none of it.
