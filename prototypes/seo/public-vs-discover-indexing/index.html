What the prototype proposes

Two tiers — separated by governance, not by effort.
Safe now · pure conformanceno architecture
  • robots: { index:false, follow:false } on the (app) group metadata
  • Self-canonical on /vendors via the existing URL builder
  • Directly realizes frozen matrix rows 8 & 10 — mints no new authority
Gated · Wave-4 & ADR-024human board
  • Middleware auth boundary for the (app) route group
  • Host-aware robots.txt + apex sitemap-index (ADR-024 Decision 9 — not a flat disallow list)
  • Structured data (e.g. ItemList) — omitted here; evaluated separately under the SEO Authority Matrix
Matrix Row 8

RFQs · quotations · workspace · account · admin — private, no public URL exists, nothing is indexed (session-gated, default-private).

Matrix Row 9

Apex sitemap-index → canonical hosts; each serves its own platform-generated robots.txt. Private app surfaces excluded by construction.

Matrix Row 10

One resource = one canonical URL; builder-only emission — no string-concatenated public URL anywhere.
